Compatability check text

All RPC calls made by the Web3.js library are defined underneath. The script will make all calls and check three things:
- Is the data sent as expected?
- Is the RPC method available to be received on the node?
- Do we receive data back from the node in the format Web3.js expects it?

When using this script to develop your own node you can check scripts.js what to send. All RPC calls are made as HTTP calls so that we can check on the flow and what to send of each call.
